`CONCISE` is implemented as an R package, which can be installed from GitHub:

#### 1. Install `devtools` if necessary
```r
install.packages('devtools')
```

#### 2. Install `CONCISE`
```r
library(devtools)
install_github("jiazhao97/CONCISE")
```

#### 3. Load package
```r
library(CONCISE)
```
