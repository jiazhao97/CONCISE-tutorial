`CONCISE` is implemented as an R package, which can be installed from GitHub by:

#### 1. Install `devtools` if necessary
```r
install.packages('devtools')
```

#### 2. Install `CONCISE`
```r
devtools::install_github('YMa-lab/IRIS')
```

#### 3. Load package
```r
library(CONCISE)
```
